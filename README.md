# Dogs Gallery

Proyecto estático desarrollado como práctica de maquetación con **HTML** y **CSS**, utilizando **Flex** **Grid** para crear una pagina responsiva inclutengo **Media Queries** y Trancisiones.

## Acceso

```
Agregar link pages
```

## Descripción

La página muestra una galeria de perros con imagenes aleatorias:

- Un menu de navegacion en la parte superior que cambia a la parte izquierda para dispositivos mas grandes.
- Un área principal con imagenes destacadas dentro de cards, estas responden al cursor del mouse.

## Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox
- Grid
- Media Queries
- Transiciones

## Características

- Layout principal construido con Grid.
- Menu navegacion creado con Flex.
- Sección principal creada con Grid, ordena el contenido de acuerdo al tamaño de la pantalla.
- Imágenes responden al cursor.
